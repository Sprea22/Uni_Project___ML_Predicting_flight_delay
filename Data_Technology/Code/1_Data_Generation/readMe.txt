-----------------------------
| Flights_Datasets_Elabs.R: |
-----------------------------
This R FILE selects the origin dataset "Flights_delay.csv", then it generates and saves the datasets 'Flights_delay_ATL_Data_Quality.csv' and 'Flights_delay_ATL_Restored.csv' (it's possible to see the dataset description in the 'Datasets' folder readMe).

-----------------------------
| Weather_Datasets_Elabs.R  |

This R FILE selects the origin dataset "Weather_ATL.csv", then it generates and saves the datasets 'Weather_ATL_Data_Quality.csv' and 'Weather_ATL_Restored.csv.csv' (it's possible to see the dataset description in the 'Datasets' folder readMe).
